21 * This class represents the access permissions to a document.
22 * These permissions are specified in the PDF format specifications, they include:
23 * <ul>
24 * <li>print the document</li>

26 * <li>copy or extract content of the document</li>
27 * <li>add or modify annotations</li>
28 * <li>fill in interactive form fields</li>
29 * <li>extract text and graphics for accessibility to visually impaired people</li>
30 * <li>assemble the document</li>
31 * <li>print in degraded quality</li>
32 * </ul>
33 *
34 * This class can be used to protect a document by assigning access permissions to recipients.

36 *
37 *
